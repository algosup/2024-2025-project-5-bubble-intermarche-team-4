# Naming Conventions

## GitHub

### Branch Names

>[!TIP]
>
>- `Good Practice:` Use a simple and descriptive name in kebab-case for the branches.
>
>```md
>documents
>pull-requests-template
>```

<br>

>[!CAUTION]
>
>- `Bad Practice:` Use a complex and/or non-descriptive name for the branches.
>
>```md
>featureindevelopment
>object
>```
